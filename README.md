# Meccha FOV Tool

## What is this?

In Meccha Chameleon, ultrawide / wide monitors (e.g. 32:9 - 5140×1440) cause the FOV to break after certain scenes (cutscenes, loading screens, etc.). This small tool continuously writes the desired FOV value, keeping the camera correct at all times.

## Usage

1. Launch the game
2. Run `MecchaFOV.exe`
3. The console displays `current fov:` and the current FOV value
4. `[1]` — enter a new FOV (max 140)
5. `[2]` — reset to default (90)
6. `[Q]` — quit

The tool continuously writes the FOV in the background with no further input needed. If you close the tool, the game will revert to the original value on the next loading screen.
